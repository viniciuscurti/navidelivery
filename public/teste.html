📋 Como Acessar a Landing Page

IMPORTANTE: Use apenas HTTP (não HTTPS) para acessar o projeto em desenvolvimento.

🌐 Links Diretos (Clique nos botões abaixo):

  • 🏠 Landing Page Principal
  • 🎬 Página de Demo
  • 💰 Página de Preços
  • 📞 Página de Contato
  • 🏢 Página Sobre

⚠️ Se os links não funcionarem:

  1. Copie e cole manualmente na barra de endereços: http://localhost:3000
  2. Certifique-se de usar HTTP (não HTTPS)
  3. Se o navegador forçar HTTPS, tente em modo privado/incógnito
  4. Aguarde 1-2 minutos para o servidor Rails terminar de carregar

✅ O que você deve ver quando funcionar:

  • Hero section com gradiente azul/roxo
  • Menu de navegação (Funcionalidades, Preços, Depoimentos)
  • Botões "Teste Grátis" e "Ver Demonstração"
  • Estatísticas em tempo real
  • Grid de funcionalidades com ícones
  • Seção de preços com 3 planos
  • Testimonials de clientes
  • Footer completo
